## vim vs VSCODE

* Vim uses less ram, is fast and typing latency is absoultely low.
* VScode is a electron bloat, but has a lot of extension, ease of use and convinience. Anyone can start coding here. And it is industry **standard**

But the  thing about vim is that LLMs queries yielded assertions that upon serious vim mastery, 32hrs of **struggle** can possibly save a lot of time in the long run.
